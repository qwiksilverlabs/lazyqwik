# Getting started

LazyQwik is a collection of utility hooks for [Qwik](https://qwik.dev/). Before continue, please make sure that you have a basic knowledge of Qwik.

## Installation

```bash
npm i @qwikgear/core
```

## Example

Just import the hook you need from `@qwikgear/core`

```tsx
import { useToggle } from '@qwikgear/core';

const [toggle, setToggle] = useToggle();
```

Visit [hooks list](/hooks) for more hooks.
